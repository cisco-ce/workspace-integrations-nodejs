Interface IntegrationConfig

Your configs for initialising your Workspace Integration. Must containt client id and secret, which you get when deploying the integraition in Control, as well as the JSON Web Token (jwt) which you get when you activate it.

  • longpolling: Your integration using long polling to get notifications.
